# 10 High-Converting UGC Template Structures (20-30s)

Use this structure for script generation and shot-list assembly.

## 1) Problem -> Solution
- Goal: Demand capture for pain-aware buyers.
- Timing:
  - 0-3s: Hook with pain.
  - 3-10s: Expand frustration.
  - 10-22s: Product as fix.
  - 22-30s: CTA.
- Prompt skeleton:
  - "Write a 24-second UGC script. Start with [pain hook], show a relatable frustration, introduce [product] as the fix with one concrete benefit, end with [CTA]."

## 2) Before / After Transformation
- Goal: Show outcome quickly.
- Timing:
  - 0-4s: Before state.
  - 4-8s: Transition moment.
  - 8-24s: After state with proof.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Create a 25-second before/after script for [audience]. Use one measurable or emotional result and mention [product] once naturally."

## 3) Social Proof Stack
- Goal: Reduce skepticism.
- Timing:
  - 0-5s: Credibility stat.
  - 5-15s: 2 short proof points.
  - 15-24s: Why it works.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Write a social-proof UGC script using [stat], [review snippet], and [benefit]. Keep it natural and not overhyped."

## 4) Objection Buster FAQ
- Goal: Convert high-intent users stuck on one objection.
- Timing:
  - 0-4s: Ask objection.
  - 4-16s: Direct answer.
  - 16-24s: Proof/example.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Write a 22-second objection-handling script. Question: [objection]. Answer clearly with one proof and a low-friction CTA."

## 5) 3-Step How It Works
- Goal: Clarity and onboarding.
- Timing:
  - 0-3s: Hook.
  - 3-22s: Steps 1-3.
  - 22-30s: Outcome + CTA.
- Prompt skeleton:
  - "Generate a 27-second UGC script with exactly 3 steps for using [product], each step under 8 words on-screen."

## 6) Comparison (Us vs Alternative)
- Goal: Win consideration stage.
- Timing:
  - 0-4s: Comparison headline.
  - 4-14s: Alternative limitation.
  - 14-24s: Your differentiator.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Write a fair comparison script, no competitor naming, showing one weak point of alternatives and one strength of [product]."

## 7) Creator Testimonial (Mini Story)
- Goal: Authentic persuasion.
- Timing:
  - 0-5s: Personal setup.
  - 5-15s: What they tried before.
  - 15-24s: Result with product.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Write in first person as a creator. Sound conversational, include one specific result, and close with [CTA]."

## 8) Day-in-the-Life Integration
- Goal: Show natural fit in routine.
- Timing:
  - 0-4s: Routine context.
  - 4-20s: Product integration moment.
  - 20-26s: Benefit in context.
  - 26-30s: Soft CTA.
- Prompt skeleton:
  - "Create a day-in-the-life script where [product] appears naturally in [routine]. Keep tone [tone]."

## 9) Offer + Urgency
- Goal: Drive immediate action.
- Timing:
  - 0-4s: Offer in first line.
  - 4-16s: Value and relevance.
  - 16-24s: Deadline/reminder.
  - 24-30s: Strong CTA.
- Prompt skeleton:
  - "Write a 20-second urgency script for [offer] ending [deadline]. Keep language ethical and clear."

## 10) Myth vs Truth
- Goal: Reframe category beliefs.
- Timing:
  - 0-4s: State myth.
  - 4-14s: Correct it.
  - 14-24s: Demonstrate truth with product.
  - 24-30s: CTA.
- Prompt skeleton:
  - "Write a myth-busting script that starts with 'Most people think...', then explains truth in plain language and connects to [product]."

## Shot-list format standard
For every generated script variant, return 4-6 beats with:
- `beat`: ordinal number
- `timeRange`: `0-3s`
- `voiceover`: spoken line
- `visual`: what creator films
- `overlay`: on-screen caption
- `intent`: persuasion purpose (`hook`, `proof`, `demo`, `cta`)
